# Enviroment

We follow VAR and CAR.

# Download

Download the vae, car and LongCLIP checkpoints.

Modify the path in `config.py`.

# Demo

Run `condition.py` to generate condition image and `demo.py` to generate the defect image.

```cmd
$ python condition.py
$ python demo.py
```
The result will be saved as `result.png`.
